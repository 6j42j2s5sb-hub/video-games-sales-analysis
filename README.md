Questo progetto analizza il dataset Video Game Sales (Kaggle) per comprendere l’andamento del mercato dei videogiochi nel tempo, le piattaforme più redditizie, i generi più popolari e le differenze di vendita tra le varie regioni del mondo.

L’obiettivo è mostrare competenze da Data Analyst attraverso:

pulizia dei dati

analisi esplorativa (EDA)

visualizzazioni con Seaborn e Matplotlib

interpretazione dei risultati

📊 Dataset
Fonte: Kaggle – Video Game Sales  
Il dataset contiene informazioni su oltre 16.000 videogiochi, con le seguenti colonne principali:

Name – titolo del videogioco

Platform – console/piattaforma

Year – anno di uscita

Genre – genere del gioco

Publisher – casa produttrice

NA_Sales – vendite in Nord America

EU_Sales – vendite in Europa

JP_Sales – vendite in Giappone

Other_Sales – vendite nel resto del mondo

Global_Sales – vendite totali

🛠️ Tecnologie utilizzate
Python

Pandas

Matplotlib

Seaborn

Jupyter Notebook

🔍 Analisi svolta
Nel notebook notebooks/analysis.ipynb vengono eseguite le seguenti operazioni:

1. Esplorazione iniziale
caricamento del dataset

analisi struttura

ricerca valori mancanti

analisi statistiche di base

2. Pulizia dei dati
rimozione righe incomplete

conversione tipi

gestione duplicati

3. Analisi esplorativa (EDA)
trend delle vendite globali nel tempo

piattaforme più vendute

generi più popolari

4. Analisi avanzata
top 10 videogiochi più venduti

confronto tra publisher

analisi piattaforme nel tempo

5. Insight finali
Sintesi dei risultati ottenuti e considerazioni sul mercato videoludico.
